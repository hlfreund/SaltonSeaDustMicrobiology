# Salton Sea Aeolian Dust Microbiome Research
The scripts included in this repo were used to process and analyze 16S amplicon sequence data and shotgun metagenome sequence data collected from Salton Sea dust. These samples were collected by members of Dr. Emma Aronson's lab in 2020 and 2021 from several study sites around the Salton Sea in Southern California. The goal of this work was to elucidate the taxonomic and functional diversity of the dust microbiome (specifically bacteria) as it relates to cyclical changes in weather within the Salton Sea region. The sequence data used for this project is available as BioProject PRJNA1157902, accessible here: https://www.ncbi.nlm.nih.gov/bioproject/?term=PRJNA1157902

You can also read our findings here: https://www.frontiersin.org/journals/microbiology/articles/10.3389/fmicb.2026.1691133/full

Many of this work was done during my PhD, and admittedly the code is not as organized as I hoped it would be - but I am actively working on cleaning up the scripts so that they could be easily understood and used.
